# SlackBot

## GitFlow Settings
For collaboration, we will be using the following settings in GitFlow
(set by running `git flow init`):
  - Production: 'master'
  - Next Release: 'develop'
  - Feature Prefix: 'feature/'
  - Release Prefix: 'release/'
  - Hotfix Prefix: 'hotfix/'
  - Support Prefix: 'support/'
